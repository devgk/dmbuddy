Introduction to WordPress

In this topic, I will introduce WordPress. It is the most powerful content management system. It powers 27% of the internet.

Introduction to WordPress

WordPress is completely free software which is used to develop dynamic websites. WordPress is very flexible and robust. Everyone from web developers to average computer users is using it to develop beautiful websites. It is completely customizable and can be used for almost anything.

WordPress is basically a blogging software. A blog is a type of website or part of a website that is maintained on a regular basis by its owner. The blog is used as a news board with entries regarding updates, commentary, reviews, opinions, and other media such as video. The term blog is actually short for "web log" but can also be used as a verb.

WordPress is completely free. There is no trial or premium. It is an open source software. It relies on the community of developers.

WordPress has two websites “wordpress.org” and “wordpress.com”.

wordpress.org

You can download WordPress software free of cost for installation on your web hosting server from wordpress.org.

wordpress.org is a site where you can download and install a software script called WordPress.

wordpress.com

“wordpress.com” provides you the WordPress hosted platform to develop a website. Wordpress.com provides hosting and ready to use installed WordPress software. It lets you get started with a new and free WordPress-based blog in seconds.

wordpress.com provides a very simple and ready to use platform for website development. With wordpress.com you do not need to go with WordPress software installation. You just need to start developing your website.

Types of WordPress websites

WordPress seems little complex. But believe me, it is very simple. Let’s get into detail. There are two types of WordPress website you can create.

  1. “wordpress.com” hosted website: This website can be created on wordpress.com.
  2. Hosted WordPress website: This website can be created using WordPress software hosted on your web hosting server.

What's the difference between Installing WordPress and wordpress.com

You can set up your website at “wordpress.com” and take your business online instantly. Creating website with “wordpress.com” is different than installing WordPress software on your server. “wordpress.com “stores all your files on their server and you need to get an account with them to set it up.

But, when you install WordPress on your server, you are bypassing wordpress.com and all storing your website data privately. This allows you to manage your WordPress website, without any limitation. This way you can unleash the full power of WordPress.
