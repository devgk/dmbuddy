Creating WordPress Page and Post

A Website is not possible with any web page. WordPress simplified the process of creating a web page using WYSIWYG (What You See Is What You Get) editor. Web pages in WordPress are termed as Page and Post. Both are technically same - just defer in functionalities. Follow these steps to create your first page and post.

Let's Get Started

Creating WordPress Page

Before starting, I recommend you to open your WordPress website. And visit wp-admin “dashboard of your WordPress website”. Login to your WordPress dashboard to access admin panel.

1

To get started adding a new page to your WordPress site, find the Pages menu in the WordPress Dashboard Navigation menu. Click Add new.

2

Create Page

  • Add the title of the page, like About, Contact etc.
  • Next is the content formatting section or post editor. This is where you’ll actually type the content of your post or page.
    If you look on the right side of the box, you’ll see two tabs. There are two modes of editing posts - Visual and Text modes.
    • Visual Tab: The Visual tab will bring up the visual WYSIWYG editor. Here you’ll see a formatting toolbar with lots of options for formatting your content. If you’re familiar with Microsoft Word or any other word-processing software, most of these icons should look familiar. Text Tab: If you click the Text tab, this will reveal a plain-text HTML version of the post editor. This version of the post editor is for editing the HTML code of your post. For most users, the Visual editor is the easiest way to write posts.
  • Preview the page to see how it will look.
    Use Publish section of the page when you’re ready to publish, you can either publish immediately, or save you page as a draft, or schedule the page to be published later.
  • After publishing you can visit this page of your website with the permalink (URL of this page).

Creating WordPress Post

Creating post in WordPress is similar to creating pages that you have read before.

1

To get started adding a new post to your WordPress site, find the Post menu in the WordPress Dashboard Navigation menu. Click Add new.

  1. Go to Posts » All Posts and delete the default “Hello world” post. To delete click on “Bin”.
  2. Similarly, go to Pages » All Pages and delete the default “Sample Page”.
  3. And finally, click on “Comments” and delete the default comment.
2

Create Post

  • Add the title of the post
    Post is like an article, update, or news. So its title should be like the heading of an article.
  • Next is to add content as you did for the page.
  • In post you will find two new options. That is categories and tags which you can assign it to your WordPress post. I will cover these two topics in more detail later.
  • Publish it similarly as you did before.
That’s it all done!
