Setup Home Page and Blog Page

Home Page is the first page of any website. It is also known as first interaction point, landing page or front page.

A WordPress website can have a dynamic blog-like front page, or a static front page which is used to show customized content.

Remember: By default, WordPress uses the first option by showing your most recent posts on the front page.

Some users prefer the second option. This feature is particularly useful for websites that want to keep a blog separate from other sections of their website or for users who want to showcase content other than blog posts on the front page of their website. It can also make the site feel less like a blog and more like a full feature website. Often this feature is used on business sites.

Create a page

Before you can set a static front page for you site, you have to create and publish the page that you would like to use as the static front page.

Create a page first that you want to use it as a home page. You can name it whatever you want. Any WordPress page can be used as static front page, but you can only select the page that has been published already.

Setting custom page as a home page

  1. The static front page setting can be configured in the Settings » Reading screen
  2. Click on the “A Static Page”
  3. Select your created page as a front page
  4. And click on save

Set Your Blog Page

You should also go back and create a second page. Just add the title of that page and don’t add any content. This page will be used to hold or show your recent blog posts. Write title of this page something like “Your Business Blog”.

  1. The blog page setting can be configured in the Settings » Reading screen
  2. “A Static Page” must be checked
  3. Select your created “Your Business Blog” page as a post page
  4. And click on save
