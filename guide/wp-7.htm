WordPress categories and tags

WordPress Categories and tags are used to organize your WordPress posts in your WordPress website or blog.

Remember: By default WordPress Categories and Tags works only for post.

WordPress categories

WordPress categories are basic ways for you to organize your posts into different categories, such as topics

Adding a New Category

If you don’t have any categories yet, follow these steps

  1. Go to Posts » All Posts
  2. Open the post you want to categories or create a new post
  3. On post editing page » Click the + Add new category link. Enter new category name and click Add new category. You’ll then see your new category has been added to the list of categories.
  4. To assign a category to a post, click the checkbox next to the category

Another way to manage post categories is through the Posts > Categories link in the WordPress Dashboard navigation menu. Clicking the Categories link will display the Categories page. From here, you can see all your categories on the right, and where to add new ones.

Category Slug

The Category Slug is the URL-friendly version of the category. It is usually all lowercase and contains only letters, numbers and hyphens.

Category Hierarchies

Categories, unlike tags, can be nested into hierarchies. For example, you might have a wildlife category, and under that have children categories such as carnivores, herbivores and omnivores animals.

Category Description

The description section is a place for you to add more details about the posts assigned to that category. It is visible on frontend of the website.

WordPress Categories vs Tags

What’s the difference between categories and tags? Think of your site like a book. Categories are like the Table of Contents and tags are like the terms in the index.

Adding a New Tag

Adding tag is very easy. Find tag section and enter your tags separated by commas. You’ll then see your new tags has been added to the list of tags.

You can also manage tags thorough Posts > Tags link in the WordPress Dashboard navigation menu. Clicking the tags link will display the tags page. From here, you can see all your tags on the right, and where to add new ones.
