Adding media in WordPress

Using a visual element like an image or photo is a great way to spruce up your post or page content.

Let’s start adding media in WordPress. You can add media in your WordPress website in several ways. The most common way is to add media using WordPress unloader, while creating page, post or custom content.

Media is a broad term used for images, videos, audios etc. Inserting media in your post or page content is just a piece of cake. But there are some limitations to keep in mind.

Media Size: Never upload large size media files such as audio and video. It may slow down your website and might make it inaccessible in low-end hosting servers.

Adding images

Images can be uploaded directly into WordPress using media unloader. Avoid uploading ultra HD images with resolution more than two thousand pixels and 500kb in size. For ultra HD images we will use third party image hosting services. The same we will do for audio and video files.
