WordPress Widgets

Widgets allow you to add different functionality and features to your WordPress sidebars without writing any code. In this article, you will understand how to add and use Widgets in WordPress.

You will see the widgets that you can add under the list of available widgets. A default WordPress installation comes with some built-in widgets like recent posts, recent comments, archives, search, etc. Later you can also install widgets according to your need.

You can add widgets to widget-ready area. These designated areas are defined by your WordPress theme. They are called sidebars or widget-ready areas.
