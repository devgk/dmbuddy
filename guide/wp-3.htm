WordPress important terms

There are few important terms of WordPress you should know before proceeding ahead.

1. Permalink

Permalinks are the permanent URLs to your individual posts, pages, as well as categories and other lists of weblog postings. In simple words - The URLs of the content you publish on your WordPress website are known as permalinks.

A permalink is what another web-logger will use to link to your article (or section), or how you might send a link to your story in an e-mail message. The URL to each post should be permanent, and never change — hence permalink.

Permalink can be found on Settings » Permalinks page in your WordPress admin area.

2. Theme

A WordPress theme is a template package used to customize the appearance of your WordPress website. A Theme modifies the way the site is displayed, without modifying the underlying software.

Themes can be found under theme menu in your WordPress admin area.

3. Plugin

A plugin is a piece of software containing a group of functions that can be added to a WordPress website. They can extend functionality or add new features to your WordPress websites. WordPress plugins are written in the PHP programming language and integrate seamlessly with WordPress.

Plugins can be found under plugins menu in your WordPress admin area.

4. Widget

Widgets are tiny blocks of specific functionality that you can place in your WordPress sidebars (also known as widget-ready-areas). This is an easy way to add things like galleries, quotes, popular posts, Facebook like boxes, and other dynamic items on their WordPress site.

Widgets can be found on Appearance » Widgets page in your WordPress admin area.

5. URL – Slug

A URL slug is the exact address of a specific page or post on your WordPress website. WordPress automatically assigns a URL slug for each new page, post etc. based on its title. It is usually all lowercase and contains only letters, numbers and hyphens.
