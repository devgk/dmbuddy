WordPress Dashboard Menu

In your WordPress dashboard, will see menu items on the left side of your dashboard. On the right side you will see corresponding options and settings page.

Here are the list of menu items

  1. Dashboard: View your WordPress websites stats
  2. Post: Used to create post
  3. Media: Used to manage web media
  4. Page: Used to create web page
  5. Comments: Used to manage website’s comment
  6. Appearance: Used to customize website’s appearance
  7. Plugins: Used to enhance the functionality of your WordPress Website
  8. Users: Used to manage your website’s users
  9. Tools: Used for migration purposes
  10. Settings: Used to manage core settings of your website.
