WordPress dashboard menu

In your WordPress dashboard, will see menu items on the left side of your dashboard. On the right side you will see corresponding options and settings page.

Here are the list of menu items

  1. Dashboard: View your WordPress websites stats.
  2. Post: Used to create and manage post.
  3. Media: Used to manage web media.
  4. Page: Used to create and manage web page.
  5. Comments: Used to moderate and manage users comment.
  6. Appearance: Used to customize the website appearance.
  7. Plugins: Used to extend and enhance the functionality of your WordPress Website.
  8. Users: Used to manage your WordPress website users.
  9. Tools: Used for advance purposes, such as website migration.
  10. Settings: Used to manage core settings of your website. Altering these settings without prior knowledge can be harmful for your website and business.

You have already learned about WordPress dashboard. You will learn about the rest of the options, as you go through these guide. If you have any doubts, connect with me on reedit.
