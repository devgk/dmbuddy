Create WordPress Post

Creating post in WordPress is similar to creating pages that you have read before.

  1. To get started adding a new post to your WordPress site, find the Post menu in the WordPress Dashboard Navigation menu. Click Add new.
  2. Add the title of the post.
  3. Post is like an article, update, or news. So its title should be like the heading of an article.
  4. Set permalink as you did for the pages.
  5. In post you will find two new options. That is categories and tags which you can assign it to your WordPress post. I will cover these two topics in more detail later.
  6. Preview the page to see how it will look.

  8. After publishing you can visit this page of your website with the permalink (URL of this page).
