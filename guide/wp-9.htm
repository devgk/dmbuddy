Create Navigation Menu

Navigation menus allow you to present a site structure to your users. They help users find information and browse through different sections of your website. We can also say that it helps you to navigate between the web pages of your website.

It is mostly visible on the header, footer, sidebar or navigation drawer. By default WordPress show navigation menu on the header of your website as you can see below.

Remember: Each theme can define their own menu locations and menu support.

Adding Navigation Menu

By default WordPress create navigation menu based on your pages. But if you want to add, remove or position any pages, then you need to create your own navigation menu.

You can add you own navigation menus in your WordPress site with a very easy drag-and-drop menu interface that allows you to create drop-down menus in WordPress.

Creating first navigation menu

  1. You can create menus in WordPress by visiting Appearance » Menus in your WordPress admin area.
    This will bring you to the Edit Menus screen which is divided into two columns. The column on your left has your pages, categories, and custom links tab. The column on the right is where you add and adjust menu items.
  2. Provide a name for your menu, e.g. ‘My First Menu’ and then click on the create menu button. This will expand the menu area.
  3. Next, you need to select the pages you want to add to the menu. You can do this by selecting the pages listed on the left hand side and clicking on Add to Menu button.
  4. After adding pages to the menu, select the theme location where you want to display the menu and click on the ‘Save Menu’ button.

Arranging Items in a Custom Navigation Menu

The menu item you added is arranged in the order you added them. With WordPress drag and drop functionality, you can easily rearrange your WordPress menu items in any order you want.

Simply drag and drop a menu item to adjust its position in the menu. As you can see below

Creating Sub-Menu

In your menu structure, drag the menu item just below the parent item. Next, slightly drag the menu item to the right. You will notice that it will automatically become a sub-menu item.

You can add multiple layers of sub-items to create deeply nested navigation menus.

Editing a Menu Item

When you add pages to your custom navigation menu, WordPress automatically uses the page title as the link text. This does not mean that you cannot change it.

All menu items can be edited by simply clicking on the downward arrow next to a menu item. Here you can change the menu’s navigational title or link text by editing navigation label.

Removing a link from Navigation Menus

You can remove a link in the same way you edit them. Just click on the downward arrow icon next to a menu item, and you will notice a red Remove link. When you clicking on it, the link will be removed from your navigation menus.

Remember: Don’t forget to click on the save menu button to store your changes before previewing your website.
